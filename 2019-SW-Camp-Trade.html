SW Camp 개발 과제(2019.10.DD)

한국무역정보통신 2019년 신입직원 SW Camp n일차 개발 과제 안내문

주제 안내

전자문서 XML 변환

  • 예시로 제시된 전자무역 관련 문서를 XML 형식으로 변환

  • 구현하고자 하는 변환 프로그램을 서술하고, 해당 내용을 프로그래밍

지시사항

  • 기본조건

    입출력 모두 UTF-8 인코딩 사용
    주어진 입출력 금지 문자 Set를 참조하여 Validation

  • 허용범위

    숫자/영어/한글/일부 특수문자 입력 가능
    한자 및 ※, ♬, ☎ 등의 2 byte 이상 특수문자 사용 불가

평가 요소

  • 변환기능 구현

    변환대상 Data Parsing 및 변환 Rule Loading 처리
    일부 반복되는 Child Element Looping 알고리즘 등

  • 소스코드 혹은 수도코드(pseudo-code)

    코드의 가독성 및 효율성
    라인별 혹은 기능별 주석 작성
    제한시간 내 동작하는 프로그램 작성 미비할 경우 수도코드로 대체

  • 프로그램 완성도

    대량의 데이터를 입력하였을 경우 처리 가능 여부
    일부 Escape Character가 필요한 특수문자의 입력 가능 여부 등
